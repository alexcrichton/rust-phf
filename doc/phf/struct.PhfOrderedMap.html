An order-preserving immutable map constructed at compile time.

Unlike a PhfMap, the order of entries in a PhfOrderedMap is guaranteed to be the order the entries were listed in.

PhfOrderedMaps may be created with the phf_ordered_map macro:

extern crate phf;
#[phase(syntax)]
extern crate phf_mac;

use phf::PhfOrderedMap;

static MY_MAP: PhfOrderedMap<int> = phf_ordered_map! {
   "hello" => 10,
   "world" => 11,
};

Note

The fields of this struct are public so that they may be initialized by the phf_ordered_map macro. They are subject to change at any time and should never be accessed directly.
